# mini-json-server-auth

📝 Requirements

### 회원가입 버튼 클릭 했을 때
- [x] 이름, 이메일, 비밀번호를 토대로 회원가입 구현

### 로그인 버튼 클릭 했을 때
- [x] 로그인 구현

---

### 로그인을 한 경우

- [x] 문구(👋🏼 준님 안녕하세요)와, 회원 정보 버튼, 로그아웃 버튼을 보여준다.

- [x] 회원 정보 버튼을 클릭하면 이메일과 이름이 출력된다 (이 때 user의 정보를 api요청을 통해 가져온다)

- [x] 로그아웃 버튼을 클릭하면 index.html페이지로 돌아오고 다시 로그인, 회원가입 버튼이 보인다,
